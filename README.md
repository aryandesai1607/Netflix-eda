# Netflix Exploratory Data Analysis

## Objective
Perform exploratory data analysis on Netflix Movies and TV Shows dataset to understand content distribution, genre trends, country-wise production and audience targeting.

---

## Dataset
- **Source:** Netflix Movies and TV shows dataset
- **Description:** Contains information about movies and TV shows available on Netflix, including type, genre, country, release year, rating, and date added.

---

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## Analysis Performed
- Data cleaning and preprocessing 
- Missing value analysis
- Distribution of content analysis
- Content growth analysis over time
- Genre-wise analysis
- Country-wise content distribution 
- Rating-based audiencd segmentation

## Key Insights
- Netflix content is dominated by **Movies** over TV shows.
- **Dramas** and **International Movies** are the most common genres.
- The **United States** leads content production, followed by countries like India, Canada and the UK.
- Netflix content is heavily skewed towards **mature audiences (TV-MA, TV-14)**.
